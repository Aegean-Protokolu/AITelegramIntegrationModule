# AI Cognitive Assistant
v2.2 (Event Bus Architecture)

A self-improving cognitive architecture designed for long-term memory, active learning, and autonomous reasoning.

> "Not just a chatbot, but an agent that studies, remembers, and corrects itself."

## System Components (The Sephirot)
*   **Decider (Keter/Malkuth):** The executive controller. Manages the cognitive loop, tools, and final decisions.
*   **Netzach (Victory/Endurance):** The silent observer. Monitors system stability, detects stagnation, and ensures persistence.
*   **Hod (Glory/Reverberation):** The reflective analyst. Critiques reasoning, summarizes history, and manages meta-cognition.
*   **Chokhmah (Wisdom):** The daydreamer. Generates raw insights and connections from latent knowledge.
*   **Binah (Understanding):** The verifier. Validates generated thoughts against grounded sources.
*   **Yesod (Foundation):** The bridge. Connects the internal system to the external world (Telegram).

## Technical Stack
*   **Core:** Python 3.10+, Tkinter (UI), Event Bus Pattern
*   **Memory:** SQLite (Storage), FAISS (Vector Search)
*   **AI Engine:** Local LLM via LM Studio (OpenAI-compatible API)
*   **Documents:** PyMuPDF (PDF), python-docx (DOCX), Semantic Chunking

Created for autonomous research and personalized assistance.
